## 🔧 Technical Specifications

###Animations Used:
```css
- fadeInUp: opacity 0→1, translateY 30px→0 (1s)
- fadeIn: opacity 0→1 (0.8s)
- float: background-position animation (20s)
- pulse: scale 1→1.1→1 (15s)
- Hover transforms: scale(1.05), translateY(-10px), rotate(360deg)
```

### Transition Speeds:
- Quick interactions: 0.3s ease
- Smooth hovers: 0.4s cubic-bezier
- Scroll reveals: 0.6s ease

### Shadow Depths:
- Light: 0 4px 15px rgba(0,0,0,0.08)
- Medium: 0 10px 40px rgba(0,0,0,0.08)
- Heavy: 0 15px 50px rgba(0,0,0,0.15-0.4)

### Responsive Breakpoints:
- Mobile: < 600px
- Tablet: 600-900px  
- Desktop: > 900px
