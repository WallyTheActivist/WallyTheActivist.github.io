# ROX Buy and Sell - Informational Website

A complete 5-page static informational website for ROX Buy and Sell, Africa's Trusted Online Marketplace.

## 🌐 Website Structure

This is a static "About Us" style website built with **vanilla HTML5, CSS3, and JavaScript** - no external frameworks or libraries required.

### Pages

1. **index.html** - Home/Welcome Page
   - Hero section with call-to-action buttons
   - Introduction to ROX and the problem it solves
   - Features preview grid
   - Full responsive design

2. **story.html** - Our Story Page
   - The founder's experience with fraud
   - Vision and key principles
   - Mission breakdown (5 numbered cards)
   - Why ROX is different from other marketplaces

3. **team.html** - About the Team Page
   - Founder profile (Cohen Ngoepe)
   - Leadership team grid with 4 members:
     - Sello Sithole - Finance Manager
     - Emmanuel Demande - Marketing Director
     - Xiluva Baloyi - Chief Operating Officer
     - OWLCLOAK - Technology Partner

4. **why-choose.html** - Why Choose ROX Page
   - Three main benefits: Safe & Secure, Simple & Efficient, Community Driven
   - Six numbered feature cards
   - Highlight boxes for key messages

5. **vision.html** - Our Vision & Join Page
   - Company vision for Africa's digital economy
   - Core values (Faith, Integrity, Innovation)
   - Contact information and CTA
   - App launch announcement

## 🎨 Design Features

### Color Scheme
- **Primary Orange**: #FF4500 (buttons, accents, headings)
- **Deep Blue**: #003366 (trust elements, navigation)
- **Gold**: #FFD700 (highlights, success elements)
- **Background**: #FFFFFF / #F5F5F5

### Typography
- **Headings**: Montserrat (Google Fonts)
  - H1: 48px
  - H2: 32px
  - H3: 24px
- **Body**: Source Sans Pro (Google Fonts)
  - Body: 16px
  - Line height: 1.6

### Responsive Breakpoints
- **Desktop**: > 900px (multi-column layouts)
- **Tablet**: 600px - 900px (2-column layouts)
- **Mobile**: < 600px (single-column, stacked layout, hamburger menu)

## 📱 How to Use

1. **Open in Browser**: Simply open any HTML file in a modern web browser (Chrome, Firefox, Safari, Edge)
2. **Navigate**: Use the navigation menu to move between pages
3. **Mobile View**: Resize your browser or open on mobile device to see responsive design
4. **Edit Content**: All content is in the HTML files - simply edit the text within the tags

## 🚀 Deployment Options

### Local Development

Upload all 5 HTML files to any web hosting service:
